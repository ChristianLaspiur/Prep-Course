Operadores lógicos

&& : Es el operador "Y" (en ingles "AND"), este operador va a evaluar si las dos condiciones se cumplen o no, para que se ejecute el código, las dos SI o SI deben ser true. 


|| : Es el operador "O" (en ingles "OR") evalua si alguna de las condiciones se cumple (true), el codigo se ejecuta.

!: Operador "NO" (en incgles "NOT") devuelve el valor opuesto al que se pasa. Ej:       !false = true;


Bucle FOR : Sirve para repetir un bloque de código "X" cantidad de veces.
